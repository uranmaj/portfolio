dogwalkers

node.js app with rest api and chat

app that makes life easy for dog owners who wish to find appropriate people to take their dogs for a walk

this app was made in a team, not everything you see is work of mine but i understand it all

Homepage

must be signed in as dog owner/dog walker/company to use the app

Dog owner pov

dog owner scrolls through the list of dog walkers choosing one they like

Chat

is established between the dog owner and the chosen dog walker
and a security code is generated for in person verification

Edit profile

users can edit their profiles to make matchmaking easier

Admin pov

admin can view and remove all users
